# 示例工程：网络、Room 与 DataStore

## 对应章节

第4章 网络、Room 与 DataStore

## 工程目标

本工程将用于配合第 4 章学习真实数据处理能力：网络请求、本地数据库、轻量配置存储和离线缓存。

## 当前状态

本工程目录先作为第 4 章配套工程入口保留。当前已明确目标、知识点和练习任务；后续可以基于前三章示例工程继续演进出完整 Android 工程。

## 计划覆盖知识点

- 网络请求基础
- JSON 与数据模型
- Retrofit / OkHttp
- Room Entity / Dao / Database
- DataStore 配置存储
- 加载中 / 成功 / 失败 UI 状态
- 本地缓存与离线展示

## 计划练习任务

- 使用假数据模拟网络返回。
- 将课程列表保存到 Room。
- 从 Room 读取本地缓存并展示。
- 使用 DataStore 保存用户偏好。
- 组合网络与缓存，实现简单离线可用页面。
