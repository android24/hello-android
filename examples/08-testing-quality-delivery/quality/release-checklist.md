# 发布前检查清单

## 构建配置

- [ ] 当前构建变体是 `prodRelease`。
- [ ] `applicationId` 符合正式包要求。
- [ ] `versionCode` 已递增。
- [ ] `versionName` 与发布计划一致。
- [ ] `BASE_URL` 指向正式环境。
- [ ] `ENABLE_VERBOSE_LOG` 为 `false`。

## 测试结果

- [ ] UseCase 单元测试通过。
- [ ] Repository 测试通过。
- [ ] ViewModel 状态测试通过。
- [ ] 关键 Compose UI 测试通过。
- [ ] 手动验收课程刷新路径。

## 风险检查

- [ ] 未新增无法解释的敏感权限。
- [ ] release 包没有测试入口。
- [ ] 日志没有输出 token、手机号、邮箱等敏感信息。
- [ ] release 变体至少在发版前构建过一次。
- [ ] 崩溃、接口错误和用户反馈有观察入口。

## 回滚准备

- [ ] 确认上一稳定版本号。
- [ ] 确认严重问题联系人。
- [ ] 确认是否可以暂停灰度。
- [ ] 确认是否有远程开关或降级方案。
