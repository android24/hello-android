# 性能体检清单

## 启动

- [ ] Application 中没有同步网络、数据库大查询或大文件读取。
- [ ] 首屏 Activity 没有做阻塞式初始化。
- [ ] 首屏 Composable 只加载首屏必要数据。
- [ ] 能记录进程创建、Activity 创建、首屏展示等关键节点。

## 卡顿

- [ ] 主线程没有执行大循环、Thread.sleep、磁盘 IO 或网络请求。
- [ ] 列表 item 有稳定 key。
- [ ] 滚动时没有频繁创建大对象。
- [ ] 图片加载、解码和缓存策略明确。

## CPU

- [ ] 计算密集任务放到后台线程或 `Dispatchers.Default`。
- [ ] IO 任务放到合适的 IO 调度器。
- [ ] 可取消任务支持取消。
- [ ] 重复计算有缓存或去重策略。

## 内存

- [ ] 不再使用的大对象能够释放引用。
- [ ] 页面退出后不会继续持有 Activity 或 View。
- [ ] 图片和列表缓存有上限。
- [ ] 可以用 Profiler 观察内存增长和释放。

## 包体积

- [ ] 检查未使用资源。
- [ ] 检查重复依赖。
- [ ] release 构建启用必要的压缩和混淆策略。
- [ ] 保存 mapping 文件，方便崩溃还原。
