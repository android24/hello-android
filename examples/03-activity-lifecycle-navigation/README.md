# 示例工程：Activity、生命周期与导航

## 对应章节

第3章 Activity、生命周期与导航

## 工程目标

本工程将用于练习 Android Activity 组件模型、生命周期日志、Intent 页面跳转、参数传递和返回栈。

## 当前状态

本工程目录先作为规划入口保留。完成第 2 章 Compose UI 示例工程后，可以继续演进为多页面示例工程。

## 计划覆盖知识点

- `Activity`
- `AndroidManifest.xml`
- 生命周期回调
- Logcat 观察生命周期
- Intent 显式跳转
- Intent 参数传递
- 返回键与任务栈

## 计划练习任务

- 创建 `MainActivity` 和 `DetailActivity`。
- 为两个 Activity 添加生命周期日志。
- 从首页跳转到详情页。
- 使用 Intent 传递课程标题。
- 按返回键观察生命周期变化。
